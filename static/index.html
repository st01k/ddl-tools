    ddl-tools

    alpha v0.9

    Designed and developed by Casey J. Murphy, with contributions of domain-specific expertise from Daniel J. Murphy

    ddl-tools provides a way to view imported .ddl/.und files in an efficiently organized manner thus expediting the troubleshooting process. Additionally, any imported .ddl/.und file can be exported as an itemized set of spreadsheet-importable .PSV files.

    The DDL Programmer's Manual was used for technical references throughout the development of this application.
